== Description ==

A map plugin for WordPress integrating Naver Maps with WordPress, powered by Shoplic. Register many places you want and display them on your Naver maps.

쇼플릭이 제공하는 네이버 지도와 워드프레스를 연동하는 플러그인입니다. 마음껏 원하는 장소를 등록하고, 지도에 표시하세요.


This plugin utilizes the Naver Maps API (https://oapi.map.naver.com/openapi/v3/maps.js) ) provided by Naver Cloud.
The information related to Naver Map API is as follows.

- Naver CLoud: https://www.ncloud.com
- Map API overview, and documentation: https://www.ncloud.com/product/applicationService/maps
- Naver CLoud AI·Naver API service terms of service: https://www.ncloud.com/policy/terms/opapi
- Naver CLoud privacy policy: https://www.ncloud.com/policy/infou/infou

이 플러그인은 네이버 클라우드에서 제공하는 지도 API(https://oapi.map.naver.com/openapi/v3/maps.js)를 활용합니다.
네이버 지도 API에 관련한 정보는 아래와 같습니다.

- 네이버 클라우드: https://www.ncloud.com
- 지도 API 안내, 문서: https://www.ncloud.com/product/applicationService/maps
- 네이버 클라우드 AI·Naver API 서비스 이용약관: https://www.ncloud.com/policy/terms/opapi
- 네이버 클라우드 개인정보 처리방침: https://www.ncloud.com/policy/infou/infou


This is an **unofficial** plugin for WordPress that offers the ability to have Naver Maps to be embeded into your WordPress frontend pages.
Please note that this is not created by, affiliated with, or supported by Naver.

이 플러그인은 워드프레스 프론트엔드 페이지에 네이버 지도를 삽입할 수 있는 기능을 제공하는 네이버 지도 서비스의 **비공식** 플러그인입니다.
네이버에 의해 제작되지 않았으며, 네이버와 제휴하지 않았으며, 또한 네이버의 지원을 받지 않았습니다.
